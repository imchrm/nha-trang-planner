## [0.5.0] — Деплой и документация
*30 мая 2026*

### Добавлено
- `vite.config.js`: `base: '/tours/nha_trang_planner/'` для деплоя в подпапку
- Папка `docs/` с файлами CONTEXT, TODO, ARCHITECTURE, CHANGELOG
- Инструкция для пользователей (`инструкция-конструктор-нячанг.md`)

### Исправлено
- Убран `<style>` блок из JSX — перенесён в `src/index.css` (шрифты, анимации, кастомные классы)
- Grid-лейаут с arbitrary values заменён на flex/block для корректной сборки Tailwind v3
- Восстановлен мобильный таб-лейаут (`sm:hidden` + sticky bottom nav)
- Лейаут на ≥640px: одна колонка (карточки сверху, расписание снизу), без side-by-side

### Изменено
- Подписи табов: `'Д.Р.'` → `'День рождения'`, `'Активные'` → `'Активные дни'`

## [0.4.0] — Персистентность и шаринг
*30 мая 2026*

### Добавлено
- **localStorage** — авто-сохранение `schedule` при каждом изменении, восстановление при загрузке
- **URL-шаринг** — кодирование расписания в base64, хранение в `#s=...` hash
- **ShareModal** — QR-код (api.qrserver.com), копирование ссылки, JSON экспорт и импорт
- **LoadConflictModal** — диалог «Два расписания» при наличии и URL hash, и localStorage (Вариант Б)
- **UrlLoadedBanner** — тост «загружено из ссылки», автоскрытие через 4 сек
- Кнопка «Поделиться» (Share2 иконка) в обоих лейаутах
- Иконки `Share2, Upload, Download, Link, Check` добавлены в импорты lucide-react

## [0.3.0] — Модальные окна и UX-защита
*30 мая 2026*

### Добавлено
- **ResetConfirmModal** — замена `window.confirm()` (заблокирован в iframe)
- **BlockingAlertModal** — три варианта текста с конкретными датами при ошибке размещения А-5 (Далат 2 дня)
- `canPlaceCardOnDay` расширен: возвращает `blockingDay` в результате для передачи в алерт
- **CardPickerModal** — выбор карточки для дня (sticky header, корректный scroll, `max-h-[85vh] overflow-y-auto`)
- Исправлен `canPlaceCardOnDay`: добавлено условие `card.weight !== 8` перед проверкой веса — двухдневная карточка теперь проходит валидацию

### Исправлено
- Кнопки «+ Добавить активность» и «+ Добавить ещё» в DaySlot теперь открывают CardPickerModal на всех размерах экрана

## [0.2.0] — Визуальные индикаторы
*30 мая 2026*

### Добавлено
- **DurationColumn / DurationIndicator** — вертикальные квадратики (7×7px) на карточке каталога: заполненные = категорийный цвет, контурные = opacity 0.2. Для weight=8: два параллельных столбика
- **Агрегированный индикатор дня** — `DurationColumn` слева от стека PlacedCardChip, показывает суммарный вес дня (цвет `#B8945A` — нейтральный янтарный)
- **IconRating** — заменил цифровой `MiniRating`: 10 звёзд (темп, цвет по интенсивности) + 10 смайликов (впечатления, цвет категории). Заполненные иконки — solid fill, контурные — opacity 0.2
- Расположение ТЕМП и ВПЕЧ. — в одну строку (`flex items-center gap-3`)

### Изменено
- `Star, Smile` добавлены в импорты lucide-react

## [0.1.0] — Базовый функционал
*30 мая 2026*

### Добавлено

**Данные:**
- 30 карточек активностей: 7 birthday (BD1–BD7), 13 active (AC1–AC13), 10 rest (RS1–RS10)
- 11 дней (d1–d11): arrival (30.05), 9 free, birthday (05.06), departure (09.06)
- Объекты `CAT` (категории с цветами) и `THEME` (глобальная палитра)

**Интерактивность:**
- Drag & drop (HTML5 API) на десктопе
- CardPickerModal (тап по дню → выбор карточки)
- DayPickerModal (тап «Добавить в день» на карточке → выбор дня)
- Система весов: `canPlaceCardOnDay()` с полной валидацией
- Двухдневная карточка А-5 (weight=8) занимает два слота

**Date-adaptive поведение:**
- 4 фазы: preparation / active / birthday / memories
- DebugPanel с пресетами дат
- Фаза birthday: CSS-конфетти (60 элементов, `@keyframes confettiFall`), баннер с shimmer, коралловый градиент фона
- Маркер «СЕГОДНЯ» с анимацией pulse на текущем дне
- Приглушение прошедших дней (opacity 0.65)

**UI-компоненты:**
- ActivityCard (collapsed/expanded, draggable)
- PlacedCardChip (карточка в слоте)
- DaySlot (header + content + capacity indicator)
- IntensityChart (гистограмма с предупреждением о 2+ активных днях подряд)
- Summary (заполнено X/9, бюджет, темп, впечатления)
- ConfettiOverlay
- Header (с декоративными SVG-элементами, фазо-адаптивный субтайтл)
- CategoryTabsComponent

**Адаптивность (первая версия):**
- ≥640px: two-column flex (карточки 40%, расписание 60%)
- <640px: таб-система с bottom nav («Каталог» / «Расписание»)

**Шрифты:** Fraunces (display) + Nunito (body) через Google Fonts
